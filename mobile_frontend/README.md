# Mobile Frontend (React Native)

This folder contains React Native equivalents of the three HTML screens from:
- `stitch_mobile_frontend/home_dashboard_raj/code.html`
- `stitch_mobile_frontend/medications_raj/code.html`
- `stitch_mobile_frontend/emergency_raj/code.html`

## Files
- `App.tsx`: simple tab state to switch between Home, Medications, Emergency
- `src/screens/HomeDashboardRajScreen.tsx`
- `src/screens/MedicationsRajScreen.tsx`
- `src/screens/EmergencyRajScreen.tsx`
- `src/theme/colors.ts`
- `src/types.ts`

## Dependency
This folder is configured as an Expo React Native app and uses Expo Material icons.

Install dependencies:

```bash
npm install
```

Start the app:

```bash
npm run start
```
